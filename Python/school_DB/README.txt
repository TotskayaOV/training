archeticture_programm.png - файлы программы в схеме(файлы БД одним блоком)

При запуске программы программа проверяет открыты ли БД и загружает их, если обращение происходит впервые.

Открывается меню выбора роли:
1. Директор
2. Учитель
3. Родитель (не включено в программу)

Директор может:
1 посмотреть список классов/учителей
2 Добавить ученика + адрес
	(при добавлении адреса, адрес сравнивается с имеющимися адресами и не записывается, в сводную таблицу к id_ученика добавляется уже существующий id_адреса)
3. Изменить данные ученика/учителя
4. Добавить оценку
5. Посмотреть оценки ученика
6. Посмотреть список учеников, их адрес
7. Добавить ученика в класс
8. Удалить ученика из класса
9. Добавить учителя
10. При выходе из программы доступен выход без сохранения данных

Учитель может:
1.увидеть список учеников своего класса
2. выбрать предмет
3. добавить оценку
4. выход осуществляется только с сохранением
